Proto.type
===


### Overview 
A good place to start prototyping.


This is a simple project for prototyping javascript projects.

### To build:

	npm install
    bower install

### To run:
	node app.js


### For node.js prototypes:

I like to use  [`nodemon`](http://nodemon.io/)
  
  	
    npm install -g nodemon
	nodemon app.js
